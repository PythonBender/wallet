Getting Started
1. Create Account The first thing you need is an account, click Create Account at the top to get started.
2. Get GO Now that you have an account, you need some GO to spend. Join our Telegram and ask for some!

NOTE: You will only get some tokens for the testnet. Mainnet has gone live in May.
3. Send Tx Now that you have some GO, share it with your friends! Click Send Tx above to send GO.
